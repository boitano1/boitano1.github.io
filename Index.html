In this class, we created interactive graphics using p5.js in open processing, using forloops, variables, and various functions. At the end of the semester, we created an interactive wireframe for a UX interface for a product or service that we created.

CSMA 111: Introduction to 3D Worlds (Transferred from a class at Moorpark College)
Alfred Borris
Fall 2020

Throughout the course of this class, we learned about the fundamentals and techniques of building 3D objects using Cinema 4D. We worked on a semester-long project where created various assets to set up an environment. With this environment, we created a story for an animation, covering how to keyframe and export animations.

CSMA 112: Interactive Prototyping
Mark Ericson
Spring 2022

Introduced creating reactive interfaces between sketch and user using Python in Processing. Utilized interactive components such as a microphone to translate color in RGB and HSV spaces. The second half of this class utilized the m5stick, experimenting with how color can be conveyed in an open space by using an UltraSonic Range Sensor.

CSMA 113: Mixed Reality
Echo Theohar
Spring 2022

Introduced the concept of AR and VR, bringing art and design into a 3D space. We created 3D interactions using assets created in other studios as well as through photogrammetry, experimenting with how a user can interact with art in a virtual world.

CSMA 214: Applied Mathematics
Michael Peguero
Spring 2022

Introduced creating algorithms for computational designs using C# within Unity. We created simulations for flocking, genetic evolution, as well as artificial life, allowing us to introudce our own variables such as spawning random objects, creating a reactive environment for these objects, and implementing our own assets. Finally, we combined these different projects to create a simulation integrating these factors.
